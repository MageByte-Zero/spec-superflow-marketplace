# spec-superflow Marketplace

Plugin marketplace for [spec-superflow](https://github.com/MageByte-Zero/spec-superflow) — a spec-first AI coding workflow plugin.

## Install

### Claude Code
```bash
/plugin marketplace add MageByte-Zero/spec-superflow-marketplace
/plugin install spec-superflow@spec-superflow-marketplace
```

### GitHub Copilot CLI
```bash
copilot plugin marketplace add MageByte-Zero/spec-superflow-marketplace
copilot plugin install spec-superflow@spec-superflow-marketplace
```

### Cursor
```bash
# Local deployment (recommended)
npx spec-superflow@latest install-cursor
```

### Gemini CLI
```bash
gemini extensions install https://github.com/MageByte-Zero/spec-superflow
```
